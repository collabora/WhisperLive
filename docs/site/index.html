Security, Observability & Scale

Production features you’d expect from a commercial API — fully open source and self-hosted.

🔑 API Key Authentication

Protect REST and WebSocket endpoints with Bearer token auth. Unauthenticated connections are rejected before any GPU resources are allocated.

🚦 Rate Limiting

Per-IP sliding window rate limiting on REST endpoints. Configurable RPM threshold with HTTP 429 responses.

📈 Prometheus Metrics

Full instrumentation — active connections, transcription latency histograms, audio processed, segments emitted, REST request counters.

🔁 Auto-Reconnect

Client-side automatic reconnection with configurable retry count and backoff delay. Only triggers on unexpected disconnects.

📖 OpenAPI Documentation

Auto-generated interactive API docs — Swagger UI at /docs, ReDoc at /redoc, and OpenAPI JSON schema.

🏗 Horizontal Scaling

Multi-GPU deployment behind nginx/HAProxy with WebSocket sticky sessions. Kubernetes-ready with Docker Compose templates.

🎓 Fine-Tune Custom Models

Train Whisper on your domain data — medical, legal, accented speech. A key open-source differentiator commercial APIs can’t match.

🐳 Docker Everything

Dockerfiles for GPU, CPU, TensorRT, OpenVINO, Edge, and Jetson. One command from zero to production.

👥 Multi-User ACL

Role-based access control (admin/user/readonly), per-user API keys with SHA-256 hashing, rate limits, and monthly quotas.

🔒 JWT / SSO

Authenticate via Keycloak, Cognito, Auth0, or Okta. RS256 JWKS and HS256 support. Combined with local API keys.

🔔 Reliable Webhooks

Async transcription callbacks with exponential backoff retry. Smart retry — no retry on 4xx, retry on 5xx and 429.

🗃 S3 / MinIO Storage

Pluggable storage backends for audio and results. S3, MinIO, or local filesystem. GDPR deletion endpoints included.

⚡ Terraform Deploy

One-command AWS deployment — VPC, ALB, ECS with GPU, S3, CloudWatch. Full infrastructure as code.

Quick Start

Up and Running in Minutes

Pick your language and start transcribing.

# Install
pip install whisper-live

# Start server
python3 run_server.py --port 9090 --backend faster_whisper --enable_rest

# Transcribe from Python
from whisper_live.client import TranscriptionClient

client = TranscriptionClient(
    "localhost", 9090,
    model="small",
    word_timestamps=True,
    enable_diarization=True,
    noise_reduction="near_field",
)

client("audio.wav")   # transcribe a file
client()              # transcribe from microphone
import { WhisperLiveClient } from './whisperlive';

const client = new WhisperLiveClient('http://localhost:8000', {
    apiKey: 'your-key',
});

// Transcribe a file
const result = await client.transcribe(audioFile);
console.log(result.text);

// SSE streaming
for await (const segment of client.transcribeStream(audioFile)) {
    console.log(segment.text);
}
import wl "github.com/collabora/WhisperLive/sdks/go"

client := wl.NewClient("http://localhost:8000", &wl.Config{
    APIKey: "your-key",
})

result, err := client.Transcribe("audio.wav", nil)
fmt.Println(result.Text)

// Check server health
health, _ := client.Health()
fmt.Printf("Status: %s (%d/%d clients)\n",
    health.Status, health.Clients, health.MaxClients)
# GPU (NVIDIA)
docker run -it --gpus all -p 9090:9090 ghcr.io/collabora/whisperlive-gpu:latest

# CPU only
docker run -it -p 9090:9090 ghcr.io/collabora/whisperlive-cpu:latest

# OpenVINO (Intel GPU)
docker run -it --device=/dev/dri -p 9090:9090 ghcr.io/collabora/whisperlive-openvino

# Edge (Raspberry Pi / ARM64)
docker build -f docker/Dockerfile.edge -t whisperlive-edge .
docker run -p 9090:9090 -p 8000:8000 whisperlive-edge

# NVIDIA Jetson
docker build -f docker/Dockerfile.jetson -t whisperlive-jetson .
docker run --runtime nvidia -p 9090:9090 whisperlive-jetson
# Transcribe a file
curl -X POST http://localhost:8000/v1/transcriptions \
  -H "Authorization: Bearer sk-..." \
  -F "file=@audio.wav" \
  -F "model=small"

# Stream with SSE
curl -X POST http://localhost:8000/v1/transcriptions \
  -H "Authorization: Bearer sk-..." \
  -F "file=@audio.wav" \
  -F "stream=true"

# List loaded models
curl http://localhost:8000/v1/models

# Health check
curl http://localhost:8000/health
